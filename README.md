# Nyx Messenger 3D Polished Website

Final polished static website for Nyx Messenger.

## Files
- index.html — main animated 3D landing page
- styles.css — responsive visual system
- script.js — canvas field, reveals, tilt, progress, mobile menu
- privacy.html, terms.html, platform-rules.html — legal pages
- assets/favicon.svg, assets/og-preview.svg

## Publish
Upload the folder to Vercel, Netlify, GitHub Pages, or any static hosting.

Before production, replace `https://example.com` in `robots.txt` and `sitemap.xml` with the real domain.

The Android button points to:
https://github.com/hanxwqweasd/12312323qwe/releases/download/v1.0.0/nyx-android.apk

Place Windows installer into `downloads/Nyx-Messenger-Setup.exe` if you want the Windows button to download directly.
